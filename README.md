# Js 3D Studio

Catálogo oficial da Js 3D Studio.

Projeto desenvolvido para exibição de produtos de impressão 3D e colecionáveis, permitindo que clientes e lojistas:

- Visualizem produtos
- Selecionem quantidades
- Adicionem ao pedido
- Enviem pedido diretamente via WhatsApp

Tecnologias utilizadas:
- HTML5
- CSS3
- JavaScript

Projeto hospedado via GitHub Pages.
